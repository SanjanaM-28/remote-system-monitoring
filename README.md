# Remote System Health Monitoring

## Description
This project monitors system metrics (CPU, Memory, Disk) from multiple clients and sends them to a central server using UDP sockets.

## Features
- Multi-client support
- Real-time monitoring
- Threshold-based alerts
- Encrypted communication

## Tech Used
- Python
- Socket Programming
- psutil
- cryptography

## How to Run

### Server
python3 server.py

### Client
python3 client.py
